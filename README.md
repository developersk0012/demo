# SK Education Website

## Files Structure
```
sk_website/
├── index.html      → Student Dashboard (main app)
├── signup.html     → Student Registration
├── login.html      → Student Login
├── admin.html      → Admin Panel
├── style.css       → All shared styles
├── shared.js       → Firebase + shared functions
├── ss.png          → Logo image (upload karo)
├── poster.png      → Home page banner image (upload karo)
└── sima.png        → Course card image (upload karo)
```

## GitHub Pages pe Host Karne ka Tarika

1. GitHub pe new repository banao (e.g. `sk-education`)
2. Ye saari files upload karo (ss.png, poster.png, sima.png bhi)
3. Settings → Pages → Source: `main` branch → Save
4. Website live ho jayegi: `https://yourusername.github.io/sk-education`

## Image Files (Tumhe Upload Karni Hain)
- `ss.png` → Logo image (circular preferred)
- `poster.png` → Home banner (1200x400px best)
- `sima.png` → Dream 465+ course card (600x300px best)
